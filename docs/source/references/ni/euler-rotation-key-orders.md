# Euler Rotation Key Orders

!!! tip
	These values are available in Lua by their index in the `ni.eulerRotKeyOrder` table. For example, `ni.eulerRotKeyOrder.YZX` has a value of `2`.

Index | Value
----- | ------
XYZ   | `0`
XZY   | `1`
YZX   | `2`
YXZ   | `3`
ZXY   | `4`
ZYX   | `5`
XYX   | `6`
YZY   | `7`
ZXZ   | `8`
