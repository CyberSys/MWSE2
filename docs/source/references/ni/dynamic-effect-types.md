# Dynamic Effect Types

!!! tip
	These values are available in Lua by their index in the `ni.dynamicEffectType` table. For example, `ni.dynamicEffectType.pointLight` has a value of `2`.

Index             | Value
----------------- | ------
ambientLight      | `0`
directionalLight  | `1`
pointLight        | `2`
spotLight         | `3`
textureEffect     | `4`
