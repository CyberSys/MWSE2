# tes3regionSound

A structure that contains region sound information.

## Properties

### `chance`

The chance the region sound will be used. Must be an integer between 0 - 100.

**Returns**:

* `result` (number)

***

### `sound`

The access to object's associated sound.

**Returns**:

* `result` ([tes3sound](../../types/tes3sound))
