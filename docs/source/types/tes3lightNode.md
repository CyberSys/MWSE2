# tes3lightNode

An attachment-capable structure that maintains dynamic lights.

## Properties

### `light`

Access to the `niLight` object mantained by this `tes3lightNode` object.

**Returns**:

* `result` ([niLight](../../types/niLight))

***

### `value`

The flicker phase of the light.

**Returns**:

* `result` (number)
